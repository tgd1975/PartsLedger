# Code of Conduct

## In short

Be constructive. Criticism of code is welcome; criticism of people is not.

## Expected behaviour

- Give specific, actionable feedback on code, inventory entries, or design decisions.
- Ask questions when something is unclear.
- Report bugs with enough detail to reproduce them.
- Submit fixes, improvements, and tests.

## Not acceptable

- Harassment or personal attacks of any kind.
- Dismissing others' contributions without explanation.
- Off-topic arguments unrelated to the project.

## Enforcement

If you experience or witness unacceptable behaviour, contact the owner
of the repository via GitHub.

## Scope

This code of conduct applies to all issues, pull requests, and discussions in this repository.
